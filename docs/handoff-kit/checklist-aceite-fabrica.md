# Checklist de Aceite para Fabrica de Software

## 1) Produto e escopo

- [ ] Backlog priorizado por valor (MVP, MMP, posterior)
- [ ] Historias com criterios de aceite objetivos
- [ ] Fluxos ponta a ponta mapeados

## 2) Contratos de API

- [ ] OpenAPI atualizado e versionado
- [ ] Endpoints principais revisados com negocio
- [ ] Schemas de erro padronizados
- [ ] Politica de versionamento definida

## 3) Dados e regras

- [ ] Dicionario de dados canonico preenchido
- [ ] Enumeracoes e validacoes documentadas
- [ ] Mapeamento SQLite -> Oracle revisado
- [ ] Regras de auditoria por entidade definidas

## 4) Arquitetura

- [ ] ADRs iniciais aprovados
- [ ] Integracoes externas identificadas
- [ ] Requisitos nao funcionais definidos
- [ ] Estrategia de observabilidade definida

## 5) Testes e qualidade

- [ ] Matriz de rastreabilidade preenchida
- [ ] Cenarios de teste por historia definidos
- [ ] Casos de borda e erro mapeados
- [ ] Evidencias de validacao anexadas

## 6) Entrega para fabrica

- [ ] Pacote de handoff consolidado em docs/handoff-kit
- [ ] Responsaveis e pontos focais definidos
- [ ] Riscos e pendencias registradas
- [ ] Reuniao de transicao realizada e ata assinada
